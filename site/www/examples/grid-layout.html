The page padding (in green) is often set first, usually on a container class. Within that container, a grid is established.
The grid isn't a "layer" like in a Sketch file, in which you place elements on top of. It's more like a "ruler" where you line containers up to along each row.
This is a 2-col container. Once you establish a container on a grid, the contents of that container (in blue) no longer have a concept of the parent grid. (in red)

This is a 3-col container. If the numbers work out nicely, you can line up a child grid to the parent grid (see example on right), but it doesn't always work out nicely. (See examples below.)
